# Clarity Social Assets

Public hosting repo for generated Clarity social media assets used by OpenClaw and Buffer.

Only publish assets that are safe to be public:

- Generated Clarity marketing images
- Exported social graphics
- Non-private demo visuals

Do not publish:

- Credentials
- Private screenshots
- Customer data
- Employer/client/work data
- Financial-plan data
- Anything not intended for public social posts
